How to use Study Cards

Navigating

  • Tap card flip to reveal the answer
  • Swipe left next card
  • Swipe right previous card
  • ← → arrow keys or Prev / Next buttons
  • Space flip card (keyboard)

Grading

  • Know it ✓ marks learned, advances to next card
  • Still learning ↩ marks for review
  • Swipe up Know it  Swipe down Still learning
  • Enable Still Learning only to drill unmastered cards

Decks & Modes

  • Choose a deck at the top to filter by subject
  • Quiz: Origin / Insertion / Action reveals one field at a time
  • Start cards flipped — quiz yourself from the answer side

Editing cards

  • Tap ✏ on any card to suggest a correction
  • Requires a GitHub account & Personal Access Token
  • Changes open a Pull Request — nothing changes without review
